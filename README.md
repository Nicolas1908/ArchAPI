## ⚙ Configuração

Atenção! É necessário que você tenha o Node.js e o Redis-server instalados!

1- Para iniciar o projeto:
> - Acesse o diretório principal do projeto
> Execute o comando: npm i

2- Para iniciar a aplicação:
> npm start
